Quick Start

Add scroll animations to any element. Content fades in when scrolling down, fades out when scrolling back up. Outro animations only trigger when content scrolls well past the top of the viewport.

<!-- 1. Include libraries --> <link rel="stylesheet" href="css/animations.css"> <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/gsap.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/ScrollTrigger.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/animejs/3.2.1/anime.min.js"></script> <script src="js/animations.js"></script> <!-- 2. Add data attributes --> <div data-anim="fade-up">Animates on scroll</div> <div class="grid" data-anim-grid>Grid with stagger</div>

Configuration

All animation values are configurable in js/animations.js. Adjust these to change the feel.

Timing Configuration
Property Default Description
scroll.start 'top 85%' When intro animation triggers
scroll.end 'top -10%' When outro animation triggers (past top)
fadeUp.y 60 Starting Y offset in pixels
fadeUp.duration 0.8 Animation duration in seconds
fadeUp.ease 'power3.out' GSAP easing function
grid.delay i * 100 Stagger delay between items (ms)
grid.duration 600 Grid item animation duration (ms)
parallax.speed 0.5 Parallax intensity (0-1)
// Override in your code AnimConfig.fadeUp.duration = 1.0; AnimConfig.fadeUp.y = 80; AnimConfig.grid.delay = (el, i) => i * 150; // Reinitialize after changes ScrollAnimations.init(); GridAnimations.initScrollGrid('[data-anim-grid]');
